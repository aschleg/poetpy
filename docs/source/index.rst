Poetpy - Python Wrapper for the PoetryDB API
============================================

Poetpy is an unofficial Python wrapper for the `PoetryDB API <http://poetrydb.org/>`_. The PoetryDB API allows users
to find a vast amount of poetry and poet data and is free to use without any authentication required. :code:`poetpy`
uses `requests <http://docs.python-requests.org/en/master/>`_ for handling the calls to the API.

Installation
============

:code:`poetpy` is most easily installed using :code:`pip`.

.. code-block :: python

   pip install poetpy
